Design decisions

  • Intended text pinned on top — raters compare against ground truth instead of guessing what the Hindi should say.
  • Blinded A/B/C/D, shuffled order per brief; model names never shown to raters (the reveal toggle below is for reviewers only).
  • Three taps per image (text verdict + authenticity), one forced choice per brief — a full session in ~13 minutes, built for Josh Jobs' mobile-first crowd.
  • Text verdict is 3-point and separate from preference, so fidelity and aesthetics are never conflated in one score.
  • Consent, age and Hindi-reading screen collected up front; non-readers routed out.

Scale on Josh Jobs

The same 20,000+ daily contributor base doing transcription QC includes native readers of every Indic script — the cheapest, fastest rater pool in the world for this eval. Gold questions (images with known-correct text) verify rater quality, mirroring Josh Talks' transcription pipeline.
